Form Event!!

The onFocus() event occurs when the input field gets focus. When the input field is focused, the border of it gets darker.
The onBlur() event occurs when the input field loses focus. When the input field is not focused, the border gets little light weighted here.
The onSelect() event occurs when the user selects some text within an input field.
The onSubmit() event occurs when the form is submitted.
The onreset() event occurs when the form is reset.
